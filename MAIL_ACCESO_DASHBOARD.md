## ¿Qué encontrará?

El dashboard le permite visualizar en tiempo real:

- **Resumen ejecutivo** — puntaje de riesgo, nivel de cumplimiento, conteo de hallazgos
- **Análisis de Pareto** — identifica las zonas que concentran el 80% del riesgo (dónde focalizar recursos)
- **Mapa de calor** — cruza zonas de proceso con normas ISO para ver dónde se concentran los problemas
- **Radar de madurez** — muestra el nivel de madurez de cada zona de su planta
- **Tabla de hallazgos** — detalle completo de cada hallazgo con búsqueda y filtros
- **Línea de tiempo** — seguimiento de acciones correctivas con plazos
